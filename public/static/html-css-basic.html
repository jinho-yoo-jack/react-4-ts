1. 입력 태그 <input>

<input> 태그는 웹에서 사용자로부터 정보를 입력받을 때 사용하는 가장 기본적인 태그입니다. type 속성을 통해 다양한 형태의 입력을 받을 수 있습니다.

주요 input 타입들

텍스트 입력 (text)

<input type="text" name="username" placeholder="사용자명을 입력하세요">

비밀번호 입력 (password)

<input type="password" name="password" placeholder="비밀번호를 입력하세요">

이메일 입력 (email)

<input type="email" name="email" placeholder="example@email.com">

숫자 입력 (number)

<input type="number" name="age" min="1" max="100" placeholder="나이를 입력하세요">

날짜 선택 (date)

<input type="date" name="birthday">

체크박스 (checkbox)

<input type="checkbox" name="hobbies" value="reading" id="reading">
<label for="reading">독서</label>
<input type="checkbox" name="hobbies" value="music" id="music">
<label for="music">음악감상</label>

라디오 버튼 (radio)

<input type="radio" name="gender" value="male" id="male">
<label for="male">남성</label>
<input type="radio" name="gender" value="female" id="female">
<label for="female">여성</label>
label 태그의 중요성: label 태그는 input 요소와 연결되어 접근성을 높이고 사용성을 개선합니다. for 속성과 input의 id 속성을 일치시켜 연결하거나, label 안에 input을 직접 포함시킬 수 있습니다.

2. <input> 외 입력 태그들

select 태그 - 드롭다운 메뉴

기본 select

<select name="city"> <option value="">도시를 선택하세요</option> <option value="seoul">서울</option> <option value="busan">부산</option> <option value="incheon">인천</option> </select>

multiple 선택 가능한 select

<select name="languages" multiple> <option value="korean">한국어</option> <option value="english">영어</option> <option value="japanese">일본어</option> <option value="chinese">중국어</option> </select>
Ctrl(Cmd) + 클릭으로 여러 항목 선택 가능

textarea 태그 - 여러 줄 텍스트 입력

기본 textarea

<textarea name="message" rows="4" cols="50" placeholder="메시지를 입력하세요..."></textarea>
textarea 속성:
  • rows: 표시될 줄 수
  • cols: 표시될 열 수 (문자 수)
  • resize: CSS로 크기 조절 가능 여부 설정

3. 양식을 만들고 값 전송하기

<form> 태그는 사용자 입력을 서버로 전송하기 위한 컨테이너입니다.

form 태그의 주요 속성

  • action: 데이터를 전송할 URL
  • method: 전송 방법 (GET 또는 POST)
  • enctype: 데이터 인코딩 방식

완전한 회원가입 폼

<form action="/register" method="POST"> <div class="form-group"> <label for="user-name">이름:</label> <input type="text" id="user-name" name="username" required> </div> <div class="form-group"> <label for="user-email">이메일:</label> <input type="email" id="user-email" name="email" required> </div> <div class="form-group"> <label for="user-password">비밀번호:</label> <input type="password" id="user-password" name="password" required> </div> <button type="submit">회원가입</button> <button type="reset">초기화</button> </form>
버튼의 종류:
  • type="submit": 폼 데이터를 전송
  • type="reset": 폼 내용을 초기화
  • type="button": 일반 버튼 (JavaScript 이벤트용)

4. 의미론적인 코드 - 시맨틱 태그

시맨틱 태그는 HTML 문서의 의미와 구조를 명확하게 나타내는 태그입니다. 검색엔진과 스크린 리더가 문서를 더 잘 이해할 수 있게 도와줍니다.

주요 시맨틱 태그

❌ 의미없는 div 사용

<div class="header"> <div class="navigation">...</div> </div> <div class="main-content"> <div class="article">...</div> </div> <div class="footer">...</div>

✅ 시맨틱 태그 사용

<header> <nav>...</nav> </header> <main> <article>...</article> </main> <footer>...</footer>

시맨틱 태그 설명

  • <header>: 페이지나 섹션의 머리글
  • <nav>: 내비게이션 링크
  • <main>: 페이지의 주요 콘텐츠
  • <article>: 독립적인 콘텐츠
  • <section>: 관련있는 콘텐츠 그룹
  • <aside>: 사이드바나 부가 정보
  • <footer>: 페이지나 섹션의 바닥글

시맨틱 태그를 활용한 블로그 구조

<header> <h1>내 블로그</h1> <nav> <ul> <li><a href="#home">홈</a></li> <li><a href="#about">소개</a></li> <li><a href="#contact">연락처</a></li> </ul> </nav> </header> <main> <article> <header> <h2>블로그 포스트 제목</h2> <p>작성일: <time datetime="2025-01-15">2025년 1월 15일</time></p> </header> <p>포스트 내용...</p> </article> </main> <aside> <h3>최근 글</h3> <ul> <li>글 1</li> <li>글 2</li> </ul> </aside> <footer> <p>© 2025 내 블로그. 모든 권리 보유.</p> </footer>

5. 문서 정보 관리하기 - 메타 데이터

메타 데이터는 HTML 문서에 대한 정보를 제공하는 데이터입니다. <head> 태그 안에 위치합니다.

필수 메타 태그들

기본 메타 태그 구성

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="웹사이트에 대한 간단한 설명"> <meta name="keywords" content="HTML, CSS, 웹개발, 튜토리얼"> <meta name="author" content="작성자 이름"> <title>페이지 제목</title> <!-- 소셜 미디어를 위한 Open Graph 태그 --> <meta property="og:title" content="페이지 제목"> <meta property="og:description" content="페이지 설명"> <meta property="og:image" content="이미지URL"> <meta property="og:url" content="페이지URL"> </head>

메타 태그 설명

  • charset: 문자 인코딩 설정 (UTF-8 권장)
  • viewport: 반응형 웹을 위한 뷰포트 설정
  • description: 검색엔진에 표시될 페이지 설명
  • keywords: 페이지 관련 키워드
  • author: 페이지 작성자
  • Open Graph: 소셜 미디어 공유 시 표시될 정보
SEO(검색엔진최적화) 팁: title과 description은 검색 결과에 직접 영향을 미치므로 의미있고 간결하게 작성하는 것이 중요합니다.

CSS 기초

1. CSS 소개 및 기본 문법

CSS(Cascading Style Sheets)는 HTML 문서의 스타일과 레이아웃을 정의하는 언어입니다.

CSS 기본 문법

선택자 { 속성: 값; 속성: 값; } /* 예시 */ h1 { color: blue; font-size: 24px; text-align: center; }

CSS 구성 요소

  • 선택자(Selector): 스타일을 적용할 HTML 요소
  • 속성(Property): 변경하고자 하는 스타일 특성
  • 값(Value): 속성에 설정할 값
  • 선언(Declaration): 속성과 값의 조합
  • 선언 블록: 중괄호 {} 안의 모든 선언들

기본 스타일링 예제

/* 모든 p 태그에 스타일 적용 */ p { color: #333333; /* 글자색 */ font-size: 16px; /* 글자크기 */ line-height: 1.5; /* 줄간격 */ margin-bottom: 15px; /* 아래쪽 여백 */ } /* h1 태그에 스타일 적용 */ h1 { color: #2c3e50; font-family: Arial, sans-serif; text-decoration: underline; }

2. CSS 문서 사용하기

CSS를 HTML에 적용하는 방법은 3가지가 있습니다.

1) 인라인 스타일 (Inline Style)

<p style="color: red; font-size: 18px;">빨간 글씨</p>

빨간 글씨

2) 내부 스타일시트 (Internal Stylesheet)

<head> <style> .blue-text { color: blue; font-weight: bold; } </style> </head> <body> <p class="blue-text">파란 굵은 글씨</p> </body>

3) 외부 스타일시트 (External Stylesheet)

<!-- HTML 파일 --> <head> <link rel="stylesheet" href="styles.css"> </head>
/* styles.css 파일 */ .green-text { color: green; font-style: italic; }
우선순위: 인라인 스타일 > 내부 스타일시트 > 외부 스타일시트 순으로 적용됩니다. 같은 속성에 대해 여러 규칙이 있을 때는 더 높은 우선순위가 적용됩니다.

3. 선택자 기본편

선택자는 스타일을 적용할 HTML 요소를 지정하는 방법입니다.

기본 선택자들

1) 태그 선택자 (Element Selector)

/* 모든 p 태그 선택 */ p { color: black; } /* 모든 h2 태그 선택 */ h2 { color: navy; }

2) 클래스 선택자 (Class Selector)

/* 클래스명이 'highlight'인 요소 선택 */ .highlight { background-color: yellow; } /* 클래스명이 'btn'인 요소 선택 */ .btn { padding: 10px 20px; border-radius: 5px; }
<p class="highlight">강조된 텍스트</p> <button class="btn">버튼</button>

3) ID 선택자 (ID Selector)

/* ID가 'header'인 요소 선택 */ #header { background-color: #f0f0f0; padding: 20px; } /* ID가 'main-content'인 요소 선택 */ #main-content { width: 80%; margin: 0 auto; }
<header id="header">헤더 영역</header> <main id="main-content">메인 콘텐츠</main>

4) 복합 선택자

/* 후손 선택자: nav 안의 모든 a 태그 */ nav a { text-decoration: none; color: #333; } /* 자식 선택자: ul의 직접 자식인 li 태그만 */ ul > li { list-style: none; } /* 여러 선택자: h1, h2, h3 태그 모두 선택 */ h1, h2, h3 { font-family: Arial, sans-serif; }

5) 가상 클래스 선택자

/* 마우스 호버 시 */ a:hover { color: red; text-decoration: underline; } /* 포커스 시 (입력 필드 등) */ input:focus { border-color: blue; outline: none; } /* 첫 번째 자식 요소 */ li:first-child { font-weight: bold; }
선택자 우선순위:
  1. 인라인 스타일 (1000점)
  2. ID 선택자 (100점)
  3. 클래스 선택자 (10점)
  4. 태그 선택자 (1점)
점수가 높을수록 우선 적용됩니다.

4. 박스모델이란?

CSS 박스모델은 모든 HTML 요소가 사각형 박스로 구성되어 있다는 개념입니다. 각 박스는 4개의 영역으로 구성됩니다.

박스모델의 4가지 구성요소

Margin (마진)
Border (테두리)
Padding (패딩)
Content (내용)

박스모델 CSS 예제

.box { /* Content 영역 */ width: 200px; height: 100px; /* Padding (안쪽 여백) */ padding: 20px; /* Border (테두리) */ border: 5px solid #333; /* Margin (바깥쪽 여백) */ margin: 15px; background-color: lightblue; }

박스모델 계산

전체 크기 계산:
  • 전체 너비 = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
  • 전체 높이 = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

box-sizing 속성

/* 기본값: content-box */ .box1 { box-sizing: content-box; width: 200px; padding: 20px; border: 5px solid black; /* 실제 너비: 200 + 40 + 10 = 250px */ } /* border-box: padding과 border 포함 */ .box2 { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid black; /* 실제 너비: 200px (padding, border 포함) */ }

5. 테두리 스타일

CSS의 border 속성을 사용하여 요소의 테두리를 꾸밀 수 있습니다.

기본 border 속성

/* 전체 테두리 */ border: 두께 스타일 색상; /* 개별 속성 */ border-width: 두께; border-style: 스타일; border-color: 색상; /* 방향별 테두리 */ border-top: 두께 스타일 색상; border-right: 두께 스타일 색상; border-bottom: 두께 스타일 색상; border-left: 두께 스타일 색상;

다양한 테두리 스타일

6. 여백 다루기

margin과 padding은 요소의 여백을 조절하는 중요한 속성입니다.

Margin (마진) - 바깥쪽 여백

/* 전체 마진 */ margin: 10px; /* 상하좌우 모두 10px */ margin: 10px 20px; /* 상하 10px, 좌우 20px */ margin: 10px 20px 30px; /* 상 10px, 좌우 20px, 하 30px */ margin: 10px 20px 30px 40px; /* 상 10px, 우 20px, 하 30px, 좌 40px */ /* 개별 마진 */ margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px;

Padding (패딩) - 안쪽 여백

/* 전체 패딩 */ padding: 15px; /* 상하좌우 모두 15px */ padding: 10px 20px; /* 상하 10px, 좌우 20px */ padding: 10px 20px 30px; /* 상 10px, 좌우 20px, 하 30px */ padding: 10px 20px 30px 40px; /* 상 10px, 우 20px, 하 30px, 좌 40px */ /* 개별 패딩 */ padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px;

여백 비교 예제

margin: 10px, padding: 0
margin: 0, padding: 20px
margin: 15px, padding: 15px

유용한 여백 기법들

중앙 정렬

.center-block { width: 300px; margin: 0 auto; /* 좌우 마진을 auto로 설정하여 중앙 정렬 */ }

마진 상쇄 (Margin Collapse)

.box1 { margin-bottom: 20px; } .box2 { margin-top: 30px; } /* 실제 두 요소 사이의 간격은 50px이 아니라 30px (더 큰 마진 적용) */
인접한 두 요소의 상하 마진은 합쳐지지 않고 더 큰 값으로 적용됩니다.

반응형 여백
